# Python Exercises

This folder contains small Python exercises that build programming fundamentals.

## Exercise Categories

- variables and data types
- strings
- lists and dictionaries
- conditionals
- loops
- functions
- files
- error handling
- small automation scripts

## Exercise Rule

Every exercise should include:

1. A clear problem statement
2. My solution
3. A short note explaining what I learned
4. Any bug or mistake I made
